# Changelog

## [2026.3.24]

### 新增
- **朋友圈卡片技能**：输入文字即可生成精美图片，6 种风格（长图、信息图、九宫格、手绘笔记、白板框图、漫画格），调用系统 Edge/Chrome 渲染，无需额外下载浏览器
- **首页快捷按钮**：新增「发朋友圈」和「PPT 制作」两个快捷入口，各含 4 个子场景模板

### 修复
- 修复快捷按钮显示技能 ID 而非中文名的问题（i18n JSON 语法错误）
- 修复调用内置技能时偶发「Unknown skill」报错
- 修复技能管理页和技能市场删除技能报「Skill not found」的问题
- 修复添加 MiniMax API Key 后连接测试报 `login fail` 的问题

### 变更
- 去除 `启动.bat`，直接双击 `UdiskAI.exe` 即可运行，程序自动识别便携模式

---

## [2026.3.22] — 首个正式版本

### 新增
- **便携模式**：解压即用，不写注册表，数据全存 `data/` 目录，支持 U 盘随身携带
- **中文文档生成技能**（cn-docx）：支持正式公文（GB/T 9704）和通用文档两种模式
- **内置办公技能**：Word 编辑、Excel、PPT、PDF、网页搜索、邮件收发、天气查询
- **技能安全扫描**：从技能市场安装第三方技能时自动进行安全审查
- **首次启动隐私协议弹窗**
- **未知错误友好提示**：API 异常时展示可读的错误说明

### 修复
- 修复会话删除后内存未释放的问题
- 修复工作目录设为磁盘根目录时报 EPERM 的问题
- 修复构建时 SKILLs 目录未正确打包的问题

### 变更
- 项目由 LobsterAI 重命名为 UdiskAI
- 移除 IM 网关、OpenClaw 引擎、自动更新等非便携功能，精简安装包体积
- 升级 Claude Agent SDK 至 0.2.81
- 内置技能重新整理，聚焦办公场景
