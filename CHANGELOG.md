# Changelog

## [2026.3.27]

### 新增
- **火山引擎 Coding Plan**：自备 Key 面板新增豆包 Coding Plan 开关，启用后自动切换到专属编码端点，与 Kimi / GLM 同等支持
- **充值自定义金额**：在线充值新增自定义金额输入，输入任意整数金额实时换算积分数（1元 = 1000积分）

### 优化
- 火山引擎提供商图标改为豆包（Doubao）品牌图标，名称显示为"豆包"
- 云端额度标题：未领取时显示"领取您的免费额度"，领取后显示"积分充值"，措辞更清晰
- DeepSeek 模型标注改用官方图标（SVG）替代文字徽章
- 下载页链接（UdiskAI.top）移至充值按钮正下方，位置更自然
- 自定义充值输入框去掉数字调节按钮，改为纯文本输入，界面更简洁

## [2026.3.26]

### 新增
- **内置云端额度模式**：无需配置 API Key，购买额度即可直接使用 AI 对话，降低上手门槛
- **一键领取奖励**：首次启动引导页面全面重设计，支持一键领取新用户体验额度
- **在线充值**：支持 ¥10 / ¥30 / ¥100 三档充值套餐，微信/支付宝扫码付款，额度实时到账

### 优化
- 充值流程改为两步确认（在浏览器完成付款 → 点击"支付完成"），避免误触关闭弹窗
- 启用云端模式时自动跳过 API 配置检查，不再提示"请先配置模型"
- 简化模型提供商列表，减少初次使用时的选择困难

### 修复
- 修复虎皮椒支付签名算法导致下单失败的问题
- 修复充值失败时无错误提示的问题

## [2026.3.24]

### 新增
- **朋友圈卡片技能**：输入文字即可生成精美图片，6 种风格（长图、信息图、九宫格、手绘笔记、白板框图、漫画格），调用系统 Edge/Chrome 渲染，无需额外下载浏览器
- **首页快捷按钮**：新增「发朋友圈」和「PPT 制作」两个快捷入口，各含 4 个子场景模板

### 修复
- 修复快捷按钮显示技能 ID 而非中文名的问题（i18n JSON 语法错误）
- 修复调用内置技能时偶发「Unknown skill」报错
- 修复技能管理页和技能市场删除技能报「Skill not found」的问题
- 修复添加 MiniMax API Key 后连接测试报 `login fail` 的问题

### 变更
- 去除 `启动.bat`，直接双击 `UdiskAI.exe` 即可运行，程序自动识别便携模式

## [2026.3.22] — 首个正式版本

### 新增
- **便携模式**：解压即用，不写注册表，数据全存 `data/` 目录，支持 U 盘随身携带
- **中文文档生成技能**（cn-docx）：支持正式公文（GB/T 9704）和通用文档两种模式
- **内置办公技能**：Word 编辑、Excel、PPT、PDF、网页搜索、邮件收发、天气查询
- **技能安全扫描**：从技能市场安装第三方技能时自动进行安全审查
- **首次启动隐私协议弹窗**
- **未知错误友好提示**：API 异常时展示可读的错误说明

### 修复
- 修复会话删除后内存未释放的问题
- 修复工作目录设为磁盘根目录时报 EPERM 的问题
- 修复构建时 SKILLs 目录未正确打包的问题

### 变更
- 项目由 LobsterAI 重命名为 UdiskAI
- 移除 IM 网关、OpenClaw 引擎、自动更新等非便携功能，精简安装包体积
- 升级 Claude Agent SDK 至 0.2.81
- 内置技能重新整理，聚焦办公场景
